# はじめに

<div id="prerequisites"></div>

## 前提

- Xcode 10 以上
- iOS 10 以上

<div id="import_sdk"></div>

## SDK の導入

### CocoaPods

`Podfile`に下記の設定を追加.

```ruby
source "https://github.com/rakuten-ads/Rakuten-Ads-iOS"

pod 'RUNA'
```

### Carthage

`Cartfile`に下記の設定を追加.

```
github "https://github.com/rakuten-ads/Rakuten-Ads-iOS"
```
